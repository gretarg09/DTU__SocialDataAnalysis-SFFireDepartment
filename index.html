Datasets

The main data set behind the analysis in this website is the Fire Calls-For-Service dataset from the city of San Francisco. The dataset includes all fire units reponses to calls. Each record includes the call number, incident number, address, unit identifier, call type and disposition. All relevent time intervals are also included in the dataset. Because the dataset is based on responses, and since most calls involve multiple units, there are multiple record for each call number. In this article the reponse time is defined as the time that it take the first responding unit to show up for service. Further analyses of this dataset can be found by clicking on the analysis button here above. The dataset can be downloaded from following link SF Call For Service . We focused on the periods from 2012 to 2017.

To get more perspective into our analysis we added information about each neighborhood in San Francisco. Unfortunately we did not find any open data sets that contained the wanted information and therefore we needed to create our own data set. The neighborhood´s information was found and extracted from this website SF neighborhood info . The website contains alot of information about each neighborhood, therefore we needed to extract the most relevant information resulting in the following list.

  • Total population
  • Population density - People per square mile
  • Area - in square miles
  • Median household income
  • Ratio of people below poverty level
  • Ratio of people that speak english
  • Ratio of foreign born residents
This info can be seen by hovering over each neighborhood in the responsive geoplot and the responsive histogram which can be found by clicking the responsive geoplot button here above.

The third data set, used in this article, contains the location and the battalion of each fire station. This data set is also self made and the information was extracted from the following website Fire Station Locations.
