Fire Department Calls for Service

Analyses of the response time of the San Francisco fire department

The project

In the following section we will analyse the dataset Fire Calls-For-Service. The dataset includes all fire units reponses to calls. Each record includes the call number, incident number, address, unit identifier, call type and disposition. All relevent time intervals are also included and that is what this article will mainly focus on. Because the dataset is based on repsonses, and since most calls involve multiple units, there are multiple record for each call number. In the following analysis the reponds time is defined as the time that it take the first responding unit to show up for service. To make the analysis even more interesting we included two additional datasets. One that includes the location of each firestation in SF and one that contains general information about each neighborhood in San Francisco.
